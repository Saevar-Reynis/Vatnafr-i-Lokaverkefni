# Vatnafræði-Lokaverkefni
Vatnasvið Hvítár, mælistöð Kljáfoss. ID númer: ID37.

# Hvaða gögn notuð?
Gögnin sem notast var við eru fengin úr íslenska gagnasafninu LamaH-Ice sem inniheldur rennslisgögn, veðurfarsgögn, og fleira.
Við skoðuðum tímabilið 1.10.1993 til 30.9.2023. Hér er slóð þar sem náð var í gögnin:
https://www.hydroshare.org/resource/705d69c0f77c48538d83cf383f8c63d6/ 

# Hvernig má keyra kóðann?
Kóðarnir eru bæði skrifaðir í Python og R. Skrár sem enda á .py eru Python skrár og skrár sem enda á .R eru R skrár. Í möppunni scripts má sjá möppu fyrir hvern lið. Í möppunum fyrir liðina má síðan finna kóðana fyrir hvern lið, 2-8. Ekki eru sér kóðar fyrir myndirnar heldur eiga myndir, og textar ef á við, að prentast út þegar kóðinn fyrir hvern lið er keyrður. Einnig er hægt að sjá myndirnar, niðurstöðutexta og töflur í möppunni figures-tables.
